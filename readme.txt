Este plugin agrega autenticación 2FA y reCAPTCHA a tu sitio de WordPress para mejorar la seguridad.

== Description ==

WP 2FA ARM es un plugin desarrollado por Oscar Brito que proporciona autenticación de dos factores (2FA) junto con reCAPTCHA para proteger tu sitio contra accesos no autorizados y bots. Este plugin está diseñado para funcionar en servidores ARM y ofrece opciones de seguridad adicionales.

== Installation ==

1. Sube el archivo del plugin a la carpeta `/wp-content/plugins/`.
2. Activa el plugin desde el menú "Plugins" en WordPress.
3. Configura 2FA y reCAPTCHA desde "Ajustes > WP 2FA ARM".

== Frequently Asked Questions ==

= ¿Necesito una cuenta de Google para usar reCAPTCHA? =
Sí, debes registrar tu sitio en [Google reCAPTCHA](https://www.google.com/recaptcha/admin/create) para obtener las claves necesarias.
